Project Overview

ModernStore is a fully functional e-commerce website built as a learning project to demonstrate cloud-based order management systems. This project showcases how modern web technologies can be used to create a complete online store with real-time cloud storage.

The website allows customers to browse products, place orders, and provides administrators with a dashboard to manage all orders from anywhere in the world using cloud storage.

Purpose of This Project

This project was created for learning and testing cloud storage systems. It demonstrates:

  • How to integrate Supabase cloud database with a website
  • Real-time order management across multiple devices
  • Admin dashboard with password protection
  • Responsive design for all devices
  • Local storage backup with cloud synchronization
  • Deploying websites using GitHub Pages

Technologies Used

HTML5 CSS3 JavaScript Supabase (PostgreSQL) GitHub Pages Font Awesome Icons

The website uses Supabase as the cloud database, which stores all orders and allows access from any device. The admin dashboard provides real-time order management with search, filter, and status update features.
